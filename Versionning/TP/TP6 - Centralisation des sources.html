Votre chef de projet vous annonce l'intégration d'un nouveau développeur.

Afin de collaborer en équipe, vous devez créer un compte GitHub (public).
Puis créez un dépôt (repository) nommé le "TP-git"

De là, paramétrez votre serveur distant "origin" au dépôt Github que vous venez de créer.
Enfin pushez votre historique sur le serveur "origin"

Depuis le poste du nouveau développeur

Vous allez utiliser le dépôt de votre voisin de droite. Sortez du dossier actuel dédié à votre dépôt.

Pour participer au projet, vous devez cloner le dépôt sur votre machine.
Puis ajoutez vos prénom et nom, au fichier "src/application.bat" (dans le header => ###)

Conservez un historique avec pour message "update header for new member"

Et pushez votre modification sur le serveur "origin".

Pour la correction du TP

Pushez l'ensemble des branches et ds tags sur le serveur "origin" avec les commandes suivantes :
git push --all
git push --tags
