Initialisation du dépot

Créez un dossier pour le nouveau projet de la structure, nommé : Versionning-TP1
Intégrez les fichiers :

  • fichier Readme.txt, composé de :
    • Titre du projet : Versionning-TP1
    • Auteur : vos Prénom Nom, promotion (nom + année)
  • fichier Licence.txt : intégrant la license  Apache 2
  • un dossier source nommé "src", composé :
    • d'un fichier application.bat (ou application.sh sous unix) composé d'un : echo "version 1"
  • un dossier binaire nommé "bin", composé :
    • d'un fichier (vide) application.exe
    • d'un fichier (vide) application.dll
    • d'un fichier (vide) application.xml
Initialisez le dépot de versionning
Exclure les binaires (dossiers ou fichiers *.exe, *.dll)
Conservez un historique avec pour message "Initial commit"

Modification à la demande du client

Le client demande que l'application précise l'heure de création à l'exécution...
Ajoutez au début du fichier src/application.bat : echo "Build : 10:00"

Conservez un historique avec pour message "Add build time"

Modification à la demande de Lead Développeur

Le lead dev a détecté un problème dans votre code source et vous demande d'ajouter un header.
Ajoutez au début du fichier src/application.bat (en personalisant avec les bonnes valeurs) : ### application.bat - 201x - votre nom ###

Conservez un historique avec pour message "Add file header"
